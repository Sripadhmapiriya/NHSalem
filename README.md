# NH Salem Sea Foods — Monorepo

A Maritime Premium seafood e-commerce platform.

## Structure

```
nh-salem-seafoods/
├── frontend/     ← React 18 + Vite + Tailwind + Framer Motion
├── backend/      ← Node.js + Express + PostgreSQL (Phase 2)
└── shared/       ← Shared types/constants (Phase 2)
```

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
